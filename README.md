# city-connectivity-graph-bfs-dfs-c
A menu-driven City Connectivity System implemented in C using Graph (Adjacency Matrix). The project analyzes city connectivity using Breadth-First Search (BFS) and Depth-First Search (DFS) traversal algorithms.

This project implements a **City Connectivity System** using **Graph Data Structure (Adjacency Matrix)** in C. It analyzes connectivity between cities using **Depth First Search (DFS)** and **Breadth First Search (BFS)**.
